## Installation

1. Clone the repository:

   ```bash
   git clone https://github.com/DanielFlockhart/Home-Hub.git
   ```

2. Navigate to the project directory:

   ```bash
   cd Home-Hub
   ```

3. Install the required dependencies:

   ```bash
   pip install -r requirements.txt
   ```

## Usage

4. To Run Program
   ```bash
      cd src
      python main.py

      Program Starting!

      ```
